# Bull Board - Job Queue Monitoring

## Access Dashboard

**URL:** `http://localhost:5000/admin/queues`

## What You'll See

- **Real-time job monitoring** for all BullMQ queues
- **Job statistics**: completed, failed, delayed, active
- **Job details**: payload, progress, logs, stack traces
- **Retry failed jobs** directly from UI
- **Clean/clear queues**

## Queues Being Monitored

- CRM Events Queue (workflow triggers, email events, etc.)
- Any other BullMQ queues created dynamically
